# 考研英语一真题正确率统计

一个简洁的考研英语一真题正确率统计工具，支持 PWA 离线使用。

## 功能

- 录入各题型正确数，自动计算得分和正确率
- 阅读理解 A 细分到 Text 1-4，精确追踪每篇表现
- 正确率三档颜色标识（绿/黄/红）
- 历年正确率趋势折线图
- 成绩数据导入/导出（JSON 格式）
- PWA 支持：可安装到桌面，支持离线访问

## 题型与分值

| 题型 | 题目数 | 每题分值 | 满分 |
|------|--------|----------|------|
| 完形填空 | 20 | 0.5 | 10 |
| 阅读理解 A（Text 1-4） | 20 | 2 | 40 |
| 新题型（阅读 B） | 5 | 2 | 10 |
| 翻译 | 5 | 2 | 10 |
| 作文 | - | - | 30 |
| **合计** | **50** | - | **100** |

## 在线使用

https://mireux-psy.github.io/english-stats/

## 数据说明

- 成绩数据保存在浏览器 localStorage 中，仅存储在本设备
- 换设备或清除浏览器数据会导致记录丢失，建议定期使用导出功能备份
- 导入功能支持合并，同年份数据会覆盖，其他年份保留

## 技术栈

- 纯前端实现，无需后端
- Chart.js 趋势图
- Service Worker 离线缓存（HTML 网络优先，静态资源缓存优先）
- Google Fonts（Noto Serif SC）字体加载

## 本地运行

直接用浏览器打开 `index.html` 即可。PWA 功能需要 HTTPS 环境。
